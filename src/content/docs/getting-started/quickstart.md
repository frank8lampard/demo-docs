GMONIT — единая платформа мониторинга приложений, инфраструктуры и бизнес-процессов с AI-детекцией аномалий.

## Что умеет GMONIT

- **APM мониторинг** — производительность приложений без изменений в коде. Java, Python, Node.js, Go и другие
- **Инфраструктура** — серверы, базы данных, очереди сообщений в реальном времени
- **Браузерный и мобильный мониторинг** — Real User Monitoring для каждого пользователя
- **AI детекция аномалий** — предиктивное выявление проблем до того как они влияют на бизнес

## С чего начать

1. Ознакомьтесь с [системными требованиями](/demo-docs/getting-started/системные-требования)
2. Установите агент для вашей технологии в разделе [Установка агентов](/demo-docs/agents/apm)
3. Настройте мониторинг по [сценариям использования](/demo-docs/use-cases)
